关于宏达路桥

宏达路桥建设有限公司成立于2003年，是一家专业从事道路桥梁工程建设的企业。公司拥有国家一级施工资质，员工500余人，其中高级工程师35人，各类专业技术人员180人。

20年来，我们承建了超过200个道路桥梁项目，总里程超过1500公里，工程质量一次验收合格率100%，优良率95%以上。
